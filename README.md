# 🏡Okaeri
### *"Welcome home."*

❤️ Shared dashboard  
📅 Calendar with notes on each day  
📝 Shared notebook  
☑️ Shared to-do list  
🎂 Important dates  
💬 Tiny message board ("Good morning!" "Good luck on your exam!")

## ❤️ Home
*The first screen your partner sees every day.*

```
Welcome home,
Ranjo ❤️ Aly

💌 Latest message
📅 Next date
🍰 Countdown to anniversary
⭐ Important dates
```
Your shared little dashboard.

## 📝 Our Room
*Everything the two of you build together.*  
Shared notes that update in real time.

Features:
- Rich text formatting
- Checklists / To-do lists
- Edit together
- Search
- Favorite notes
- Invite partner via code

## 🌙 My Corner
*A private space.*  
Nobody else can read it.

Perfect for:
- Gift ideas
- Date planning
- Random thoughts
- Draft love letters
- Personal reminders

## 📅 Shared Calendar
*A planner for your life together.*

Every day can contain:
- A quick note
- A planned date
- A schedule
- A reminder

Example:
```
7:00 PM
Dinner at our ramen place

9:00 PM
Watch that movie we kept postponing
```
It behaves like a note, just organized by time. 

## ⭐ Important Days
*Special dates automatically appear on the dashboard.*

Examples:  
❤️ Anniversary  
🎂 Birthdays  
✈️ Vacation  
🎓 Graduation  

Instead of digging through the calendar, your important memories are always waiting for you.

## 💌 Message Board
**A tiny wall for everyday affection.**

Each partner has one message.  
Writing a new one simply replaces the previous one.

```
Aly
"Good luck today.
I'm proud of you."

──────────────

Ranjo
"I'll be home late.
Don't wait for dinner."
```

Simple. No chat history. No pressure. Just the latest thought your partner wanted to leave for you. 

## 🔒 Privacy
Every note and calendar entry can be either:
```
🏡 Our Room  
Shared between both partners.

🌙 My Corner  
Visible only to its owner.
```
One tap changes where it belongs.

## 💡 Design Philosophy
**Okaeri** isn't meant to replace messaging apps.  
It's meant to become the place couples quietly return to every day.  
To remember anniversaries, plan dates, leave little notes, keep promises, and slowly build a shared home, one page at a time.
